The Lost Compass
Anno Domini MDCCCXXXIV — Navigator’s Instrument
For the hand behind the curtain
What It Is

The Lost Compass is a prop instrument built around a QMC5883 magnetometer and a TS90D continuous-rotation servo. The servo drives a physical needle on a printed compass dial; the firmware tracks its position using a dead-reckoning model, calculating how far the needle has turned from known speed constants and elapsed time. The device runs as a standalone WiFi access point served by an ESP32-C3 microcontroller — no internet connection is required. Four operational modes govern the needle’s behaviour, switchable in real time from any phone or tablet on the network.

Modes of Comportment
Normal
Free Needle
Tracks genuine magnetic North via the onboard sensor. The needle moves as the prop is turned or handed between players. A calibrated offset corrects for the sensor’s mounting angle within the housing.
Fixed
Fixed Bearing
The needle locks to a bearing you set — entered as degrees (0–359) or a cardinal direction. It holds that bearing regardless of how the prop is moved, tilted, or handed to another player.
Search
Lazy Spin
Slow, sweeping arcs with smooth easing. Suggests an uncertain or awakening instrument — the compass is seeking rather than knowing. Useful for first contact with an artefact or ambiguous navigation.
Disturbance
Erratic Spin
Rapid, unpredictable bursts of movement. Conveys a cursed, overwhelmed, or magically interfered instrument. The mechanical jerkiness is intentional and physical — it will be heard as well as seen.

First Commission — Calibration
  1. 1 Power on. Open the sliding base lid on the bottom of the compass and move the power switch to On. Place the compass on a flat surface, then open the top lid.
  2. 2 Connect. Join the CompassControl network on a phone or laptop. Open http://192.168.4.1/config in a browser.
  3. 3 Face North. Using a phone compass as reference, orient the prop so it points to magnetic North — the hinges and dome of the lid should face North.
  4. 4 Tune the heading offset. Optional — only required on first assembly or if the sensor has been remounted. The config page shows a live compass rose. Use the ±5° and ±15° buttons to bring the virtual needle to 0°. Adjust the software needle only — not the physical one. The offset is stored in flash and need not be repeated.
  5. 5 Home the physical needle. Rotate the physical dial needle by hand to point to the N mark on the compass face.
  6. 6 Initialize. Press Initialise Dial on the config page. This tells the firmware the physical needle is at 0°, resetting the internal position tracker. The instrument is now commissioned.
  7. 7 Operate. The compass starts in the Fixed Bearing mode so navigate back to the main control page and select the mode you want it to run in upon opening the lid.
On sleep: closing the lid holds the needle position in memory — allow a few seconds for the network to restart on wake before reconnecting. A full power cycle requires only steps 5–6; the heading offset is stored in flash and survives restarts.
